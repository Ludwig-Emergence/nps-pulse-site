# NPS GO — NPS Pulse 首发产品

**一句话定位：** NPS Pulse 是一家专注 NPS 方法论的客户满意度咨询公司，面向 TOC 企业/组织，提供三层产品矩阵：

| 产品 | 阶段 | 定位 |
|------|------|------|
| NPS GO | 0→1 | 低成本、快速入门、基础满意度管理工具（首发产品） |
| NPS PLUS | 1→N | 复杂体系、长期满意度管理 |
| NPS Counseling | 独立 | 单项咨询项目，按需定制 |

## 基本信息

- **公司名称：** NPS Pulse
- **产品名称：** NPS GO
- **项目发起人：** Lu

- **启动时间：** 2026-07-05
- **当前状态：** 🟡 规划中

## 组织架构

```
Lu（总经理）— 决策 & 检核
  └─ Arnie（总经理助理）— 协调 & 执行 & 汇总
       ├─ 产品经理（product-design）— NPS GO 产品分析文档 & PRD
       ├─ 市场调研（market-research）— 每日业务洞察 & 竞品情报
       └─ 市场运营（marketing-ops）— 获客渠道搭建 & 传播执行
```

### 各组职责

**产品经理**
- 用户场景与痛点分析
- NPS GO 功能定义与 PRD 编写
- 盈利模式设计
- 依赖市场调研组提供数据支撑

**市场调研组**
- 每日洞察交付（行业动态、竞品变化、用户反馈）
- 市场战略评估（市场状态、行业潜力、机会与风险）
- 为产品经理提供调研数据
- 为总经理提供决策依据

**市场运营组**
- 获客渠道搭建（小红书、官网等）
- 内容策略与传播执行
- 品牌定位与视觉
